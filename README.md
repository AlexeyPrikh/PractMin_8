# Myflask App in Docker

Работа с докером

## Создания образа Docker

Сборка образа:

`make build`

## Запуск контейнера

Запуск контейнера на основе подготовленного образа:

`make start`

## Остановка контейнера

`make stop`


## Отчистка

Запуск контейнера на основе подготовленного образа:

`make clean`

## Проверка состояния

`make healthcheck`
